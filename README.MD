# Atlas Custom Server 4x4
This can be used as a template for anyone looking to setup a 4x4 atlas server on Nitrado (https://server.nitrado.net/usa/rent-gameserver). Currently the map only has islands in two of the four grids.

# Server setup guide
https://atlas.gamepedia.com/Server_setup

# Map editor
Use this tool to add/remove islands to your liking. More info here:
- https://github.com/GrapeshotGames/ServerGridEditor
- https://atlas.gamepedia.com/Private_server_editing
- https://wiki.nitrado.net/en/Server_Grid_Editor
